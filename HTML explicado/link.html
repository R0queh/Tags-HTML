Tag <a></a> :

A que vem do inglês anchor(ancora), dentro da tag teremos o "href="..."" (Hiperlink REFerence) onde eu coloco meu link desejado, para links internos do meu site podemos só usar o nome dos arquivos html e caso não se encontre na mesma pasta que o atual projeto basta você mostrar o caminho(diretório) todo que deve ser seguido assim como nas imagens, caso você va usar links externos deve ser usado todo o link: Ex:"http://..."
Além disso entre as tags nós podemos colocar um nome para esse hyper link.

Obs: Precisa abrir e fechar a tag.

Estrutura:

<a href="(diretório/link desejado)">nome que vai ficar para o link</a>
