Tag <h></h> :

Do inglês header que significa cabeçalho que vai do h1-h6 sendo 1 o maior e 6 o menor usamos para escrever titulos na nossa pagina pois ficam em destaque.

Obs:Precisa abrir e fechar a tag.

Tag <p></p> :

Usada para execultarmos paragrafos possuem um espaçamento padrão que serve para separar os paragrafos e deixar uma organização muito melhor.

Obs:Precisa abrir e fechar a tag.

Tag <br> :

Vem do inglês break ela é usada para quebrar a linha e mandar a outra parte para a linha de baixo. Podemos usar sempre que quisermos trocar a linha

Obs:Não precisa fechar a tag.

Tag <hr> :

Horizontal Rule ou régua horizontal forma uma linha na horizontal para separação. Podemos usar sempre que quisermos adicionar uma linha horizontal.

Obs:Não precisa fechar a tag.
