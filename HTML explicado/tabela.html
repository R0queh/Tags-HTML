tag <table></table>:

Vem de table que significa tabela em inglês e junto com essa tag usamos mais alguns parâmetros como:

border=:

vem de border em inglês que significa borda, usamos para definir o tamanho da borda da tabela e é necessário para que se desenhe a tabela que estamos pedindo

width=:

Vem de width em ingles que significa largura e nós podemos definir para determinar um tamanho fixo para nossa tabela

Tag <tr></tr>

vem de table roll que significa linha de tabela, é realmente cada linha e precisaremos usar essa tag toda vez que quisermos uma linha nova.

Tag <td></td>

Vem de tables data que significa dados da tabela, são os dados de cada linha e podemos tambem defini-las como as colunas da tabela.

Tag <th></th>

Vem de table header que significa cabeçalho de tabela, e usamos para darmos o titulo de cada culuna da tabela

dentro de tais tags podemos colocar alguns itens:

imagens :

usando a tag img(caso deseje saber como usa-la clique aqui).

links :

com a tag a (caso deseje saber como usa-la clique aqui).

colspan :

Vem de (collum span) e significa expansão da coluna.

rowspan :

Vem de (row span) e significa expansão da linha.

Estrutura:

<table border=(tamanho da borda desejado) width=(tamanho da borda desejado) >
<tr>(linhas da tabela)
<th rowspan=""><
/th>(título da tabela)
</tr>
<td>(dados da tabela)
</td>

</td>

</td>

</td>

</table>
