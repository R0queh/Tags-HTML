tag <form></form>:

usamos para fazermos formulários tudo o que estiver dentro dessa tag será enviado para onde desejamos

Tag <input></input>:

Significa entrada e vai ser utilizado sempre que queremos alguma entrada de dados no nosso formulário dentro dessa tag temos informações como :

type:

vai ser o tipo de dados que iremos aceitar nesse input
EX:
text para entrarmos com arquivos de texto.
password para podermos colocar senhas de maneira que o texto fique escondido.
radio para criarmos formulários com opções onde clicamos na bolinha desejada, e nesse caso se precisamos que somente uma informação possa ser selecionada por formulário damos o mesmo nome para as opções possiveis.
button para criarmos botões e dentro deste podemos usar o atributo value para determinarmos o que estará dentro do botão.
checkbox para criarmos uma caixa de verificação.
submit para enviarmos as informaçoes para uma banco de dados.

Name:

o nome daquela entrada de dados.

Tag <select></select>:

vem de select que significa seleção, usamos para criarmos uma seleção entre os itens que você programou.

Tag<textarea></textarea>:

Vem de textarea que significa área de texto usamos para criar uma caixa de texto maior para escrever o tanto que o usuário deseja.

Estrutura:

<form>
<input type="(typo de entrada de dados)" name="(nome que queremos para essa entrada de dados)">
</form>
para fazer um menu de seleção:
<select name="Estado">
<option value="sp">SP</option>
<option value="mg">MG/<option>
<option value="rj">RJ</option>
<option value="rs">RS</option>
</select><br>
Para fazer uma area de texto:
<textarea></textarea>
